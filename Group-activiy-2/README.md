## Activity 1: 

1. 
- Extract `part2-notes-part2-6.zip` and  `cd part2-notes-part2-6`
- Run `npm install`

2. When the installation is finished
- Open a new terminal
- run `npm run server`

2. 
- Open another terminal
- Run `npm start`
- Visit:
  - http://localhost:3000/
  - http://localhost:3001/notes/

3. Connect to the end points in `actions.rest`.
